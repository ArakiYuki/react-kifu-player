# react-kifu-player

`react-kifu-player` は、Reactアプリケーションにおいて将棋の棋譜を美しく再生・表示するためのモダンなUIコンポーネントライブラリです。

## 特徴

- **フルReact実装**: 状態管理とUI描画がReactのパラダイムに完全に則っており、既存のReactプロジェクトへの統合が極めて容易です。
- **TypeScript完全対応**: 全てのAPIが型定義されており、開発時の補完や静的解析が強力に効きます。
- **ヘッドレスUIによる柔軟性**: 状態管理（`useKifuPlayer`）とビュー（`ShogiBoard`, `ControlBar`等）が分離されているため、独自のUIを自由に構築できます。
- **モダンなデザイン**: 「木目調」から「ガラス調」まで、多彩なテーマを標準搭載。現代のWebアプリケーションに馴染む美しいデザインです。
- **高度な機能**: 評価値グラフ（`EvalGraph`）、候補手リスト（`CandidateList`）、次の一手問題（`useShogiProblem`）など、単なる棋譜再生にとどまらない高度な機能を提供します。

## 対象読者

- 自分のWebサービスに将棋盤を埋め込みたい開発者
- 棋譜の解析結果や評価値グラフをモダンなUIで表示したい方
- Reactベースで独自の将棋アプリ（次の一手問題アプリなど）を開発したい方
